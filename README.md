## Contributing
1. Fork this repository
2. Clone your forked repository
3. Run `npm install` (or `yarn install`)
4. Run `npm run dev` (or `yarn dev`)
5. Make your changes
6. Commit your changes
7. Push your changes
8. Open a pull request
